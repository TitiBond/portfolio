mes projets

Fish Memory

Petit projet en Python d'un jeu de mémoire basé sur le Memory. L'idée était de mélanger cela avec des images d'un jeu de société que j'adore "Twin it!". Nous avons eu 2 jours pour réaliser ce projet.

Dungeon and Dragon

Ce projet est un D&D like textuel. Il a été réalisé en JAVA afin de nous permettre d'apprendre le POO. Nous avons réalisé ce projet en 4 jours.

Quiz Android

Quiz Android est un petit projet que nous avons réalisé en 2 jours. Nous avons utilisé JAVA. Vraiment interessant de découvrir la création d'application smartphone. Notre semaine sur Android n'a duré que 4 jours, j'aurais adoré avoir plus de temps !

Calculator

J'ai eu envi de pratiquer html/css et Javascript. J'ai donc décidé de faire cet exercice connu de la calculette. Une petite journée de travail pour terminer. J'ai pris beaucoup de plaisir à réfléchir à la logique du script :)

La caverne des jeux

La caverne des jeux est un projet réalisé dans le cadre du cours de PHP à la Coding Factory. L'objectif était de réaliser, en 3 jours, un site PHP dynamique, sans framework, avec connexion à une base de données MySQL. Pour la partie Front, le framework Bootstrap est utilisée. Le projet réalisé correspond à un site marchand de jeux de société.
